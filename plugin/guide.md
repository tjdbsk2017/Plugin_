## 웹개발 과정 (프론트엔드 + 백엔드)

# 프로젝트를 위해 필용한 선택지

```
1. 서버선택
2. 백엔드 언어선택
3. 데이터베이스 선택
4. 아키텍트 만들기
5. 프론트엔드 (SPA, 링크방식)
```

# 웹개발 기술사항

```
1. OS 선택 (window server, linux(추천))
    - window
    - centos (추천, 취업)
    - ubuntu (추천, 개인) - hanpixel
2. 웹서버 (웹언어를 해석할 수 있게 도와주는 역할이기도 하고 서버의 역할을 함)
    - apache (취업 추천)
    - nginx (떠오르는 샛별) - hanpixel
    - node (자바스크립트) - socket io ( 채팅 프로그램 만들기 좋음 )
3. 백엔드
    - php - hanpixel
    - python (추천) - hanpixel
    - java & jsp (취업)
    - node - hanpixel
    - go lang
4. 데이터베이스
    - oracle DB
    - mysql & mariadb (취업) - hanpixel
    - mongodb (NoSQL)
5. 개발방식
    - restful api (취업) - hanpixel
    - MVC (model - controller - view)
6. 프론트엔드
    - HTML, CSS, JS
7. 프론트 방식
    - SPA (react(상 - hanpixel), angular(백엔드지식), vue(하))
    - 페이지이동 방식
8. ui/ux
    - 반응형
    - 하이브리드 (웹 + 모바일웹)
    - 네이티브 (웹 + 앱)
```
